# DoWhile2021
![Dowhile logo](https://github.com/Mars079/DoWhile2021/blob/main/mobile/src/assets/logo.svg)<br>
<hr>

DoWhile is a fullstack project developed during the NLW Heat (2021) event by the [Rocketseat](https://www.rocketseat.com.br/) team for learning purposes. 
The project allows users to share their thoughts about the upcoming [event](https://dowhile.io/inscricao) through a light, interactive app.

## Built using:

<ul>
 <li> TS </li>
 <li> Node </li>
 <li> React </li>
 <li> React Native </li>
 <li> Vite, Expo </li>
 <li> Sass </li>
